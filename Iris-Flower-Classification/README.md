# 🌸 Iris Flower Classification

## 📌 Project Overview

This project focuses on classifying iris flowers into three different species using Machine Learning. The classification is based on the measurements of sepal and petal dimensions.

The dataset used is the famous **Iris Dataset**, which is widely used for beginner-level classification problems.

---

## 🎯 Objective

To build a machine learning model that can accurately predict the species of an iris flower based on:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## 🌼 Dataset Information

The dataset contains 3 classes of iris flowers:

* Setosa
* Versicolor
* Virginica

Each sample has 4 features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
