# 🎬 Movie Rating Prediction

## 📌 Objective
The objective of this project is to build a Machine Learning model that predicts movie ratings based on features such as genre, director, and actors.

---

## 📊 Dataset
The dataset contains information about movies, including:
- Genre
- Director
- Actors
- Rating (Target Variable)

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🔄 Steps Performed
1. Data Collection  
2. Data Preprocessing  
   - Handling missing values  
   - Extracting main actor from multiple actors  
   - Encoding categorical variables  
3. Feature Selection  
4. Model Training using **Linear Regression**  
5. Model Evaluation  

---

## 🤖 Model Used
- Linear Regression
