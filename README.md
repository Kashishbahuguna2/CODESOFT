# 💻 CodeSoft Internship Projects

Welcome to my GitHub repository for the CodeSoft Internship.

## 📌 Projects Included

### 🌸 1. Iris Flower Classification

* Built a classification model to identify iris flower species.
* Used features like sepal length, sepal width, petal length, and petal width.
* Applied Logistic Regression and Random Forest algorithms.
* Achieved high accuracy (~95%+).

---

### 🎬 2. Movie Rating Prediction

* Developed a model to predict movie ratings based on features like genre, director, and actors.
* Applied data preprocessing techniques such as label encoding.
* Used machine learning algorithms for prediction.
* Focused on handling categorical data effectively.

---

### 🚢 3. Titanic Survival Prediction

* Built a model to predict whether a passenger survived or not.
* Used features like age, gender, passenger class, etc.
* Performed data cleaning, handling missing values, and feature engineering.
* Applied classification algorithms like Logistic Regression.
