# 🚢 Titanic Survival Prediction

## 📌 Objective
The goal of this project is to build a Machine Learning model that predicts whether a passenger survived the Titanic disaster or not.

---

## 📊 Dataset
The dataset contains information about passengers such as:
- Age
- Gender
- Passenger Class (Pclass)
- Fare
- Embarked Location

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🔄 Steps Performed
1. Data Collection
2. Data Cleaning (handling missing values)
3. Encoding categorical variables
4. Feature Selection
5. Model Training (Logistic Regression)
6. Model Evaluation

---

## 📈 Model Performance
- Accuracy: **~79%**

---

## 🔍 Insights
- Female passengers had a higher survival rate.
- Passengers in higher classes (Pclass 1) had better survival chances.
- Fare and age also influenced survival.

---

## ✅ Conclusion
The Logistic Regression model performed reasonably well with an accuracy of around 79%. This project demonstrates the basic workflow of a Machine Learning model.
